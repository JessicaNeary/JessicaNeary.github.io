What is grid based design?

Grid based design is where you break the design of a webpage down into a grid to make it easier to style. Each box in the grid can then be turned into a html tag and styled individually in css.

Whats all the hype about responsive webpages?

Responsive webpages change the elements on the page depending on the browser window. This means the design can adapt to still look good and easy to read on smaller screens, such as for mobile.

What is semantic structure?

Semantic structure is where elements are named in a way that makes it clear what it is to both the browser and the developer. For example the 'table' tag is clearly selecting a table element, while the 'div' tag is not clearly referring to anything. By giving div a class or ID it can become semantic, eg div class='header'
