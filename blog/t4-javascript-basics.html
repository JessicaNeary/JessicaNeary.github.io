Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. Consider the example of cleaning the windows. Control flow is the order in which you execute the cleaning steps, in a logical fashion from the first step to the last. Explain the difference between accessing data from arrays and object literals. When accessing data from an object you use the property name to reach the data, i.e: objectname[propertyname] or objectname.propertyname. A function is a block of code that performs a specific task when executed. Functions save you repeating code when you want the same thing to happen more than once and let us do the same thing to multiple objects with multiple parameters.
