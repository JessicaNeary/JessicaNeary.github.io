Javascript Syntax and Use

Parantheses ()

1. Used to execute a function, e.g: myFunction ( )
2. Used to control order of operations, e.g: 2 * ( 3 + 5 )
3. Used to denote function parameters, e.g: myFunction ( para1, para2 )
4. Used to set conditions of for, while and if statements, e.g: if ( thing == true ) { do this }

Brackets []

1. Used to access object properties, e.g: object [ property-name ]
2. Used to access a specific cell in an array, e.g: array [ 2 ]

Braces {}

1. Used to denote the code to be executed when a function is called, e.g: function () { code to execute }
2. Used to create an object, e.g: var object = { }
Can also be used this way to create objects with existing properties by placing the properties inside the braces, e.g: var object = { height: 20 }

Single quotes ' '

Used to indicate a string, e.g: 'This is a string.' A string is a sequence of characters.

Double quotes " "

Same as single quotes. It is up to the programmer (or the project standard) which quotes to use. If quoting something inside a string it is useful to use both to save escaping the inner quotes, e.g: "this is a 'quote' inside a string"
